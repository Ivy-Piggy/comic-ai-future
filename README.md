# AI冲击下 · 漫画的重新优势

当AI重构了整个内容生产流程——自动生成剧本、图片生成视频、用工作流建立项目——漫画专业学生的核心优势在哪里？

本项目系统性地探讨这一问题，从漫画专业学生的既有优势出发（漫画技能、圈层连接、艺术审美），结合AI的真正能力（自动剧本、图生视频、工作流编程），描绘五条可行的未来进化路径。

## 六大板块

| 板块 | 内容 |
|------|------|
| 序言 | 技术地震下的漫画专业处境 |
| 维度对比 | 漫画专业 vs AI 能力矩阵分析 |
| 漫画优势 | 三大不可替代优势深度分析（技能/圈层/审美） |
| AI优势 | AI三大能力领域详解（剧本/视频/工作流） |
| 出路 | 五条未来进化路径（A-E） |
| 案例与思考 | 前沿实践与五句话建议 |

## 技术栈

- 纯静态HTML/CSS/JS网站
- Comic × Cyberpunk 视觉风格
- Canvas粒子动画（Ben-Day点×赛博霓虹）
- IntersectionObserver滚动驱动动画
- GitHub Pages 自动部署
